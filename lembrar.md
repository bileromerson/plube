# brainch

main = codico final

develop = funcoes novas acabadas mas nao incrementadas

feature/name = funcao especifica em desenvolvimento que nao ta pronta: e criada a partir do brainch develop; Vercao estavel

hotfix/name = correcao de coisas criticas no meio de producao: e criado a partir do brainch main; vercao publicada. **quando estiver pronta para ser publicada deve ser  juntada ao develop tambem**

release/name = reune todas as alteracoes do brainch develop + ajustes para mandar para deploy: ajustar vercao, change log, deixar preparado para baixar, sem arquivos inuteis. tambem ajusta qual e a proxima release para mandar para deploy
